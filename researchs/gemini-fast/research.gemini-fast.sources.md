##  Referanslar ve Kaynaklar (Sources)

Bu teknik rapor, Gemini 1.5 Flash modeli tarafından aşağıdaki yüksek performanslı kodlama standartları baz alınarak oluşturulmuştur:

* **High Performance Python Regex:**
    [Google Developers - Python Regular Expressions](https://developers.google.com/edu/python/regular-expressions)
    *Büyük veri setlerinde regex optimizasyonu için Google mühendislik standartları.*

* **Luhn Algorithm (ISO/IEC 7812):**
    [GeeksforGeeks - Luhn Algorithm Implementation](https://www.geeksforgeeks.org/luhn-algorithm/)
    *Kredi kartı doğrulaması için kullanılan matematiksel algoritmanın en hızlı Python implementasyonu.*

* **Efficient File I/O (Streaming):**
    [Real Python - Reading Large Files in Python](https://realpython.com/read-before-you-write-python/#reading-large-files)
    *Bellek dostu (Memory-efficient) dosya okuma ve 'yield' kullanımı teknikleri.*
